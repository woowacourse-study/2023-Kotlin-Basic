# 2023-Kotlin-Basic
코틀린으로 돌아보는 우테코

### :pencil: How to study
- repository를 fork하여 미션을 진행합니다.
- 스터디 repository에 본인의 브랜치를 만듭니다.
- 본인 브랜치로 진행한 미션의 PR을 보냅니다.
- 리뷰어가 PR에 대해 코드리뷰를 합니다.
- 코드 리뷰가 반영되고 나면 리뷰어의 판단에 따라 merge합니다.
