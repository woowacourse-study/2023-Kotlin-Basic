# kotlin-lotto
로또 미션 저장소

## 기능 구현 목록

- [ ] 구입 금액을 입력받는다.
  - 음수일 수 없다.
  - 1,000으로 나누어 떨어져야한다.
- [ ] 수동으로 구매할 로또 수를 입력 받는다.
  - 음수일 수 없다.
- [ ] 수동 개수만큼 로또 번호를 입력 받고, 저장한다.
  - 각 로또 번호는 ', ' 로 구분된다. 
  - 각 로또 번호는 1 ~ 45 범위에 존재한다.
  - 로또 번호는 6개이다.
- [ ] 구입금액에서 수동으로 구매한 로또 금액을 제외한 만큼 자동 로또 번호를 생성한다.
- [ ] 당첨 번호와 보너스 번호를 입력받는다.
  - 각 로또 번호는 ', ' 로 구분된다.
  - 각 로또 번호는 1 ~ 45 범위에 존재한다.
  - 로또 번호는 6개이다.
  - 보너스 번호는 하나이다.
- [ ] 당첨 통계와 수익률을 계산하여 출력한다.
